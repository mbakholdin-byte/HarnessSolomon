# Installation on Windows

:::info Work in progress
This page is being written. Check back soon.
:::

Prerequisites:

- Python 3.12+ (from python.org or winget)
- pip 24.0+
- Docker Desktop or WSL2 with Docker

Installation:

```powershell
pip install solomon-harness
```

For development install with extras:

```powershell
pip install solomon-harness[dev,test]
```
